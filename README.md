# AssetFlow AI

Monorepo root for the AssetFlow AI project.

- `frontend/` — the complete React + TanStack Start application (landing site, login, and the full asset management dashboard). Ready to run.
- `backend/` — reserved for the backend service. To be added by the backend team.

## Frontend quick start

```bash
cd frontend
npm install   # or npm install
npm run dev   # or npm run dev
```
